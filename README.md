# arch-updater

**arch-updater** is a precision update utility for Arch Linux users who refuse to treat system updates as a "black box." Unlike standard update wrappers, this tool prioritizes manual inspection and granular decision-making, ensuring you know exactly what enters your system and why.

## Overview

In the spirit of the Arch Way, this utility is built for those who want to maintain full control over their environment. It provides a transparent interface to audit package changes, read upstream documentation, and selectively apply updates only when you are ready.

## Key Capabilities

* **In-depth Inspection:** View version jumps and technical diffs before committing to an update.
* **Changelog Integration:** Access official release notes and upstream changelogs directly to identify new features or potential breaking changes.
* **Selective Upgrading:** Full control to include or exclude specific packages from the current update cycle on the fly.
* **Designed for Power Users:** A streamlined workflow for those who value system stability, manual oversight, and total transparency.
